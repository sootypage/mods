
MoneyMod (Forge 1.20.1)

This is a starter project including:
- Bank system (capability placeholder)
- Coins (copper/silver/gold)
- Commands: /bal, /pay, /deposit, /withdraw
- Shop system (JSON persistence)
- Transaction logging

NOTE:
You still need to run Forge MDK setup (gradlew genIntellijRuns or Eclipse equivalent)
and hook classes into your mod main class.

This is a template — not a compiled mod jar.
